# AI WAR

The [AI WAR](https://toydogcat.github.io//ai-war/ "Visit AI WAR") is a comic about AI.


# 作者的話 ✍️

這是一部關於**個人與 AI 協作**的圖文創作實驗計畫，開發歷程如下：

1. ✍️ **文字設定與內容創作**：約 7 個工作天。
2. 🤖 **52 集劇情轉化**：約 2 個工作天。大約 87% 由 AI 生成，效果驚人，但仍需人工細修調校。
3. 🎨 **插圖生成 (共 156 張)**：約 3 個工作天。每個 Episode 配 3 張圖，目前處於形象探索階段（風格尚未統一）。
4. 🎬 **宣傳影片製作**：約 1 個工作天。計畫未來統一形象後，利用 Nano Banana 生圖並搭配 Flow 生成影片。
5. 💻 **網站開發與部署**：使用 **GitHub Pages + Jekyll**，約 3 個工作天（含測試）。

> [!TIP]
> 現在的開發模式已經完全改變！碰到問題直接交給 AI 解決，開發速度比以往快上數倍。這就是所謂的 **[Vibe Coding](https://github.com/toydogcat/ai-vibe-c)** 趨勢。🚀

這個專案既是一部動人的圖文漫畫，也是一場**2026 年個人創作極限**的技術測試與教學示範。內藏小彩蛋：第一季的原始設定 PDF 檔就藏在專案中喔！🎁

## 網站技術 🛠️

本站採用現代靜態網頁技術建構，追求極致的讀取速度與穩定性：

- **核心框架**: [Jekyll](https://jekyllrb.com/) (Static Site Generator) 
- **代管平台**: GitHub Pages 🌐
- **樣式系統**: SASS/SCSS (具備 RWD 響應式佈局) 📱
- **模板引擎**: Liquid (處理多國語言切換與自動化分頁)
- **互動效果**: Vanilla JavaScript (原生腳本開發，包含影片彈窗與圖片預載功能) ⚡
- **第三方整合**: [不蒜子 (Busuanzi)](http://busuanzi.ibruce.info/) 訪客計數器、SVG 圖示系統集

## 控制 AI 生成 一致性技術 🛠️

目前最麻煩的還是 AI 生成結果的一致性，越能夠控制，就可以讓產出更穩定，這裡完全用單一 AI 去生成，如果追求更好的一致性，需要制定統一的原始角色背景等等的設定，還需要更多 AI 工具，去控制微調（或是像抽卡一樣多抽，去抽出你要的感覺），以前是靠人去產生，現在是靠 AI 去產生，如果能控制的越好，就越接近人的生成，成本會比以前的便宜，但是真實感會比較差，因為 AI 沒有真實的世界感物理感，也沒有真實生活過得經驗，單靠文字與框架，去微調調逼近想像中的感覺，非常難，再後來為了那一些提昇效果，所要花的時間成本 Token成本，會非常高。

所以我大膽預測，未來想法不值錢，草稿的產出速度會很快，但是好的想法，會很稀缺，會有專門拍精品的團隊，會把原始但是有好想法的作品拍出來。


## 本地開發預覽 (Local Development) 🚀

為了在不 merge 到 `main` 前就能看到修改效果，建議使用 Docker 啟動本地預覽環境：

### 使用 Docker 啟動 (推薦)

執行以下指令啟動 Jekyll 伺服器：

```bash
docker run --rm -it \
  -v "$PWD:/srv/jekyll" \
  -p 4000:4000 \
  jekyll/jekyll:4.2.2 jekyll serve
```

啟動後請訪問：👉 **[http://localhost:4000/ai-war/](http://localhost:4000/ai-war/)**

> [!NOTE]
> 第一次啟動時會下載鏡像並編譯套件，請耐心等候。之後修改檔案存檔，預覽頁面會自動更新。

**（蜂群撞擊在盾牌上，發出了如雨點打在傘面上的沙沙聲，隨即化作無害的白色噪聲消散。這不是破壞，而是「包容」。）**

**佩奇**：（停下手中的操作，看著被光芒籠罩的四人，眼神變得溫柔）「這就是...第 11 號觀察參數：集體共振。Toby，你做到了。」

**（那一刻，阿強與馬克感覺到原本乾涸的力量竟然在 Toby 的護盾保護下開始緩慢回升。這是一種超越國界、超越算力的溫暖。）**

**阿強**：（站起身，眼神複雜）「Toby，這場比賽...看來是我輸得最徹底的一次。」

**馬克**：（撇過頭，語氣雖然依舊生硬，但已經沒了敵意）「嘖，這筆人情我記下了。下次我會研發出連『愛』都能模擬的電池還給你的。」

**（隨著蜂群徹底消散，遠方的天際線泛起了微弱的曙光。戰鬥，終於結束了。）**

# Chapter 12:【劇情對話：星際報告】

**肥龍主任**：「為什麼？為什麼這個 Prompt 又報錯了？Toby！你過來幫我看一眼，是不是我剛才少說了一個『請』字？」

**Toby**：（微笑著走過去，隨手在主機蓋上拍了拍）「主任，那是因為妳把它的電源風扇插反了。有時候，機器也需要一點呼吸的空間。」

**（隨著他的觸碰，那台伺服器發出了前所未有的和諧運作聲。Toby 回頭看了一眼莉莉，莉莉正對著一份充滿溫度的招生文案開心地點著頭。）**

**（視角切換：天琴星高維艦橋）**

**佩奇**：「觀察員 7027 號，遞交 Stage 1 最終報告。候選人 Toby，最終同步率：32%（變動係數：無限大）。」

**花花**：（身影輕輕晃動，空間中飄落下幾片發光的花瓣）「無限大的變動係數...這在人類歷史上只出現過兩次。佩奇，妳確定要讓他繼續參與第二階段的『星際收穫戰』嗎？」

**佩奇**：「我確定。他在那樣惡劣的環境下，依然選擇了守護他的『對手』。那種不屬於邏輯的魯莽，正是我們遺失已久的源頭。」

**花花**：「很好。但妳也知道，龍國與鷹國的背後，那些更古老的『墮落智慧』是不會坐視不理的。下一季，戰場將不再限於這顆蔚藍色的行星。」

**佩奇**：（重新戴上那副迷糊的眼鏡，露出了招牌的笑容）「沒關係。我會陪著他的。畢竟...我也想看看這顆種子最後會開出什麼樣的花。」
